uniapp学习篇（ps:花了两个月的时间边做项目边学习）

配置相关

pages.json文件

配置页面路由，也可在相对应的页面中配置标题颜色，背景色，是否启用下拉刷新等等

启动模式配置，仅开发期间生效，用于模拟直达页面的场景，如：小程序转发后，用户点击所打开的页面。 在 App 里真机运行可直接打开配置的页面，微信开发者工具里需要手动改变编译模式

样式与布局

1.uni-app目前仅支持px和%,做适配时宽度使用%，高度使用px即可;
2.页面元素宽度在uniapp中的换算计算公式： 750px 元素在设计稿中的宽度 / 设计稿基准宽度。eg: 设计好宽度为640px， 元素A在设计稿上的宽度为100px,那么元素A在uniapp里面的宽度应为：750 100 / 640。
3.样式导入：使用@import导入,用;表示语句结束。eg: @import ‘../../**.css’。

生命周期

1.onLoad:监听页面加载，其参数为上个页面传递的数据，类型为Object(仅在页面第一次加载时触发,后面不再触发该周期函数)
2.onShow:监听页面显示
3.onReady:监听页面初次渲染完成
4.onPullDownRefresh:监听用户下拉动作
5.onReachBottom:页面上拉触底事件处理函数
6.onshareAppMessage:用户点击右上角分享 微信小程序
7.onPageScroll: 监听页面滚动
8.onTabItemTap:当前是tab页时,点击tab时触发

模板语法与数据绑定

1.与vue保持一致
2.class与style:

3.事件及事件绑定:事件对应表：左侧为Web，右侧为uniapp

导航及页面传参

1.navigator标签跳转
2.js跳转:注意：跳转到 tabBar 页面只能使用 switchTab 跳转
(1)uni.navigateTo:保留当前页面，跳转到应用内的某个页面，使用uni.navigateBack可以返回到原页面。

(2)uni.redirectTo:关闭当前页面，跳转到应用内的某个页面。

(3)uni.reLaunch：关闭所有页面，打开到应用内的某个页面。

(4)uni.switchTab：跳转到 tabBar 页面，并关闭其他所有非 tabBar 页面。

(5)uni.navigateBack:关闭当前页面，返回上一页面或多级页面。可通过 getCurrentPages() 获取当前的页面栈，决定需要返回几层。

网络请求

图片选择及预览

1.uni.chooseImage：从本地相册选择图片或使用相机拍照。

2.uni.previewImage：预览图片

文件上传、下载

1.上传：uni.uploadFile.将本地资源上传到开发者服务器，客户端发起一个 POST 请求，其中 content-type 为 multipart/form-data。

2.下载：uni.downloadFile.下载文件资源到本地，客户端直接发起一个 HTTP GET 请求，返回文件的本地临时路径。

数据缓存

1.uni.setStorage：将数据存储在本地缓存中指定的 key 中，会覆盖掉原来该 key 对应的内容，这是一个异步接口。

2.uni.getStorage：从本地缓存中异步获取指定 key 对应的内容。

3.uni.getStorageInfo：异步获取当前 storage 的相关信息。

4.uni.removeStorage：从本地缓存中异步移除指定 key。

5.uni.clearStorage：清理本地数据缓存。

6.uni.setStorageSync：将 data 存储在本地缓存中指定的 key 中，会覆盖掉原来该 key 对应的内容，这是一个同步接口。

7.uni.getStorageSync：从本地缓存中同步获取指定 key 对应的内容。

8.uni.getStorageInfoSync：同步获取当前 storage 的相关信息。

9.uni.removeStorageSync：从本地缓存中同步移除指定 key。

10.uni.clearStorageSync：清理本地数据缓存。

下拉刷新

onPullDownRefresh：在 js 中定义 onPullDownRefresh 处理函数（和onLoad等生命周期函数同级），监听该页面用户下拉刷新事件。需要在 pages.json 里，找到的当前页面的pages节点，并在 style 选项中开启 enablePullDownRefresh。当处理完数据刷新后，uni.stopPullDownRefresh 可以停止当前页面的下拉刷新。

上拉触底

onReachBottom: 在js中定义onReachBottom处理函数，监听该页面用户上拉触底事件。
